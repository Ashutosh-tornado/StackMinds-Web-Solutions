# StackMinds Redesign — Prototypes (NON-PRODUCTION)

These are **standalone design prototypes** for brainstorming the site redesign.

- They are **completely isolated** from the live app. They do **not** import from `src/`,
  do **not** touch `index.html`, and are **not** part of the Vite production build.
- Each file is self-contained HTML (Three.js + GSAP loaded from CDN). Just **double-click
  to open in a browser** — no build step, no dev server needed.
- Safe to edit or delete anytime. Nothing here affects stackminds.in.

## Files
- `home-v1.html` — Homepage concept: dark premium look, 3D WebGL hero, "Build · Market · Grow"
  positioning, real portfolio, GSAP scroll animations.

_Once we're happy with a direction here, we port it into the real React app deliberately._
